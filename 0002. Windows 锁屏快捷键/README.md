## 🤖 AI - Windows 锁屏快捷键

在 Windows 操作系统中，你可以使用以下快捷键快速锁定屏幕：

- **Windows徽标键 + L**：这是锁定屏幕的最快方式。按下这个组合键后，你的电脑会立即转到锁屏界面，要求输入密码或其他验证信息才能重新进入系统。
